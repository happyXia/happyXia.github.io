记一些平时用到又不会的 linux 命令

grep

用法

  • 可以用来搜索文件中的指定字符串
  • 可以用在别的命理后面做筛选用

参数

  • -c: 只输出匹配行的行数。
  • -h：查询多文件时不显示文件名。
  • -l: 不区分大小写（只适用于单字符）。
  • -n：显示匹配的行和行号。
  • -s：不显示 不存在或无匹配文本 的错误信息。
  • -v：方向匹配，即显示不包含匹配文本的所有行。
  • -R：递归查询，即连同子目录中的文件一起查询。

df

用法

  • 查看硬盘硬盘信息

参数

  • -h：让各个数字变得可读性更好，例如把1024个字节输出成 1k

pwd

用法

  • 查看当前路径
