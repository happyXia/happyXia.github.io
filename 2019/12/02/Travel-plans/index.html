2019.12.新加坡

计划

原计划7天的新加坡之旅，为了提高每天的旅行质量并控制费用缩短为5天4夜，因此舍弃掉了一部分可不去的景点，只保留了几个精挑细选🤓出来的高质量景点。包括新加坡动物园、滨海湾金沙酒店、新加坡国立大学、新加坡环球影城这些又有趣又有文化（逼格）的地方。

对于航班的选择经过调查和对比，认为直航还是挺必要的。换乘虽然便宜，但是太折腾了，本来只要5个小时的航程换乘一下至少11个小时，特别是有一些半夜的换成航班，只能在机场过夜，严重影响休息和旅行体验。而且新加坡不是很远，直飞和转机的机票差的不多。所以虾选择了上海到新加坡直航往返里面的最便宜的一组（以后去远的地方还是会考虑转机的，欢第一次出国以体验为主哈），24号下午3点40从浦东T2起飞，晚上10点到新加坡樟宜机场（用google地图查了一下新加坡地铁运营到晚上11点多），到酒店11点刚好洗洗睡觉准备第二天的旅行。返程是28号的晚上11点，29号的早上4点到浦东，可以睡5个小时，记得带上眼罩，头枕和耳机。

然后是关于退税。新加坡的消费税是7%，但是扣除服务费后，退税税率基本是5.5%左右 。可寻找贴有“退税”标识的商家或者在同一店铺消费满100（同一零售店累积最多3张 同日收据/发票）新元以上即可享受退税。只有出境时完全未使用的的实体商品才可以退税。然后记一下3种退税方式（其实只有人工和自助机器两种，但是人工退税也分eTRS退税和退税表退税，所以我还是分了三种方式）。

  • 人工 eTRS (Electronic Tourist Refound Scheme) 退税：
    • 购物时：在eTRS零售商店购物时，用同一张信用卡/借记卡作为退税卡，以标注所有购物记录。离开时，向商店索要eTRS退税票、原始发票/收据。
    • 离境前：
      • 在办理登机手续之前（在离境关口之前），在eTRS自助服务台就开据了eTRS票据的商品及服务申请退税。
      • 如需将消费品托运，请先在位于机场离境厅（办理离境手续前）或游轮码头一楼的 eTRS 自助服务台使用您的凭证或 eTRS 票据申请退还消费税。千万不要将可退税商品托运了再去海关，他们没法查验商品，就无法获得退税。一定是先海关查看后再办托运！
      • 如欲随身携带消费品，请携带您的消费品前往机场离境转机厅（办理离境手续后）或游轮码头二楼，于 eTRS 自助服务台申请退税。
  • 退税表退税：
    • 购物时： 若消费商家没有eTRS标志，即不属于eTRS系统，则请商家提供退税表，填写完整并签字。消费后请保存原始发票/收据。
    • 离境前：持有退税表，前去海关检查处(Customs Inspection Counter)办理，需出示消费品、始发票/收据、登机牌、机票确认单。海关会大致目视你买的东西是否与单据相符，然后在单据上盖章。在海关检查处盖章后，将表格投至机场邮箱中。表格将送至相应的商家进行处理。
  • 自助机退税：
    • 有些店铺不用自己填退税单，把护照给店员，他们会给你打一张退税的小票，在机场退税区有自助的机器，就可以自己办理退税。

樟宜机场有3个航站楼，每个航站楼都可以退税，主要流程是这样的：申请退税——办理登机手续——托运行李——出关——退税——免税区购物——安检——登机。

退税就是先在这个自助机器上申请退税单GST Refound。

不懂的话，工作人员会帮你。出关后找到这个柜台办理退税。

Day 1（12-24） 🛫:

中午从上海出发赶路为主，晚上到达新加坡。

我们寄几🚙去机场，不停浦东机场的地下停车室（停5天要500多），停机场附近的停车场，有24小时免费接送的那种，虾预定了24号到29号5天一共109块大洋，比我们两个人来回大巴贵不了多少，而且我们回来凌晨4点也没有大巴和地铁，如果不自己开车去就得打车回，血亏。所以我们寄几开车去停机场外的停车场是最好的选择，带行李也方便。

  • [12 : 30 - 13 : 10] - 家 -> 停车场
  • [13 : 10 - 14 : 00] - 停车场 -> 浦东T2
  • [14 : 00 - 15 : 40] - 值机、安检、等飞机
  • [15 : 40 - 21 : 55] - 上海浦东 -> 新加坡樟宜 （航班号 HO1605）
  • [22 : 00 - 23 : 00] : 机场 -> 酒店

Day 2（12-25）🎠:

“游乐园” 主题日。

第一站环球影城。新加坡环球影城拥有多项顶尖独特的游乐设施与景点，融汇电影主题的崭新项目及表演。环球影城24个景点和双轨过山车等项目中，有18个专为新加坡设计，全球独有的游乐项目，包括科幻影集太空堡垒(Battlestar Galactica)为蓝本，高度达42.5米的双轨过山车（虽然我们应该不会坐😂）；以及拥有3500个观众席、全球最多座位的未来水世界剧场等各种游乐项目。《神偷奶爸》中的格鲁、露西和姑娘们与一群淘气的小黄人全都闪亮登场欢庆佳节，芝麻街邻居和来自马达加斯加的狂野小伙伴很想和你来个美丽邂逅，史上最激动人心的降雪盛况等你尽情体验。只限12月2日至2020年1月1日。

然后看一下鱼尾狮塔（如果有的话，不过据说已经开始拆除或者整修了）

接下来是圣淘沙天际线斜坡滑车（据说好玩又便宜），斜坡滑车及空中吊椅引进自新西兰，放眼亚洲，目前只有日本和新加坡才有。在英比奥山顶坐上Luge，从12层楼的高度沿着650米的蜿蜒跑道往下滑冲，记得抓稳驾驶杆，享受乘风滑行的舒畅快意。同时Luge车上装有独特刹车器，方便游客自行控制速度，安全无虑。滑行抵山脚之后，可乘坐Skyride返回起点。

最后是傍晚时候的《时光之翼》表演。时光之翼拥有阵容强大的歌手和世界级的音响系统，是世界上唯一设置在海上的固定演出。每天傍晚，圣淘沙的海之颂音乐喷泉便会海滩上展开缤纷夺目的焰火、喷水和激光效果的精彩表演。而且无论下雨或天晴，这项表演都会在每天晚上如期举行，每场演出约25分钟。

  • [00 : 00 - 8 : 00] - 8点起床
  • [8 : 00 - 10 : 00] - 一边找早饭一边向环球影城出发，到环球影城预计要40分钟
  • [10 : 00 - 16 : 00] - 环球影城游玩，午餐在环球影城内解决
  • [16 : 00 - 17 : 30] - 去 Skyline Luge Sentosa 玩滑车和缆车
  • [17 : 30 - 19 : 30] - 找地儿吃晚餐
  • [19 : 40 - 20 : 40] - 《时光之翼》
  • [20 : 40 - 21 : 40] - 圣淘沙 -> 酒店

Day 3（12-26）🐘:

今天的主题是 “animals!”

白天逛新加坡动物园 (Sigapore Zoo)，世界十大动物园之一。园内以开放概念为设计，利用热带森林与湖泊等天然屏障代替栅栏，使游客可以不受铁笼和铁柱的遮栏而看得一清二楚，各类动物在没有人为屏障的舒适环境下过著自由自在的生活，与游客和平共处，有些温顺的动物可以零距离接触。

晚上逛夜间野生动物园 (Night Safari)，世界上第一个专为夜间动物而设立的野生动物园，园区内居住着超过2500只动物。可乘坐游览车，穿梭于世界7个不同地理区域，从崎岖的喜马拉雅山麓到亚洲河区森林的沼泽河岸。游览车无法到达的地方，游客可徒步在雨林中蜿蜒的步行道上，与花豹、渔猫、马来大狐蝠、沙袋鼠等近距离接触。有动物的互动表演《夜晚的精灵》，以娱乐方式配合舞台灯光呈现夜间动物特有的自然习性。

  • [00 : 00 - 8 : 00] - 起床
  • [8 : 00 - 9 : 00] - 吃早饭并前往克拉码头，酒店到克拉码头预计16分钟
  • [9 : 00 - 10 : 30] - 坐新加坡河游船，从新加坡河上参观整个新加坡
  • [10 : 30 - 12 : 00] - 在克拉码头吃午饭，有辣椒螃蟹（据说珍宝海鲜的🌶🦀特别有名）或者肉骨茶（看上去感觉是排骨汤😂）
  • [12 : 00 - 13 : 00] - 吃午餐的地方 -> Singapore Zoo
  • [13 : 00 - 18 : 00] - xxhh’s 新加坡动物园 happy time
  • [18 : 00 - 19 : 30] - 觅食时间
  • [19 : 30 - 22 : 30] - xxhh’s night safari
  • [22 : 30 - 23 : 00] - 肥酒店

Day 4（12-27）🎡:

今天的重点是 “Downtown”。包括大名鼎鼎的滨海湾金沙酒店，空中花园，鱼尾狮公园，新加坡摩天观景轮，以及新加坡国家博物馆和新加坡滨海湾花园。

这些景点就不一一介绍了，都是能感受到整个城市氛围的一些好地方。

  • [00 : 00 - 8 : 00] - 起床
  • [8 : 00 - 9 : 00] - 吃早饭，然后把行李拿到金沙酒店，不能开房的话可以先放前台
  • [9 : 00 - 10 : 00] - 从金沙酒店出发，经过双螺旋桥，逛到新加坡摩天观景轮
  • [10 : 00 - 11 : 00] - 到摩天轮上看新加坡，摩天轮一圈45分钟（最新通知，摩天轮在维修，要到1月1号才开，所以可能改成新加坡艺术科学博物馆）
  • [11 : 00 - 12 : 00] - 找午饭吃，然后前往新加坡国家博物馆
  • [12 : 00 - 13 : 30] - 参观新加坡国家博物馆
  • [13 : 30 - 14 : 00] - 新加坡国家博物馆 -> 鱼尾狮公园
  • [14 : 00 - 14 : 20] - 在鱼尾狮公园找没有人的 corner ~~ 📷
  • [14 : 20 - 15 : 00] - 鱼尾狮公园 -> 滨海湾花园（步行18分钟，途径滨海艺术中心音乐厅可以和大榴莲拍个照所以多预留了点时间）；如果还没办入住就想去酒店办入住，酒店就在滨海湾花园边上，办好入住之后可以上空中花园（就是那个屋顶上的船）看一看。
  • [15 : 00 - 18 : 00] - 游览滨海湾公园的花穹和云雾林，然后晚饭吃花园里的小麦
  • [18 : 00 - 22 : 00] - 回酒店，可以在酒店商场里的逛街，也可以去屋顶无边泳池感受一波，一览新加坡绝美屋顶泳池的景色。

Day 5（12-28）🛬:

今天是本次xxhh欢乐旅行的最后一天，虾决定用 ”university + hiking + shopping“ 来结尾。

首先是新加坡国立大学。感受一下新加坡的一流大学，随便在校园里吃个午饭。然后徒步穿过步行圣地南部山脊到达怡丰城购物+吃晚饭。南部山脊 (The Southern Ridges)，是一个具有历史价值的区域，也孕育了自然界最珍贵的一些动植物。可将新加坡的壮美风景尽收眼底。沿途青葱的环境、开阔的空间让这里成为了徒步与单车骑游的理想场所。它拥有 360 度俯瞰城市的视野，是饱览新加坡、海港与南部岛屿全景的最佳地点。步行桥中最为独特的莫过于连接花柏山公园和直落布兰雅山公园的亨德森波浪桥（Henderson Waves）。它被誉为新加坡最高最美步行桥。 然后怡丰城是新加坡最大的购物中心，坐落于圣淘沙岛(Sentosa Island)对面的滨水之地。买完东西就肥酒店提行李然后准备溜溜去机场咯。

  • [00 : 00 - 8 : 00] - 起床
  • [8 : 00 - 9 : 00] - 洗洗刷刷吃早饭然后去前台办理退房并寄存行李
  • [9 : 00 - 10 : 00] - 滨海湾金沙 -> 新加坡国立大学 NUS
  • [10 : 00 - 12 : 00] - 逛大学然后在大学里找午饭吃
  • [12 : 00 - 17 : 00] - 步行走南部山脊
  • [17 : 00 - 19 : 00] - 怡丰城逛街吃晚饭逛街
  • [19 : 00 - 20 : 00] - 回金沙酒店拿行李
  • [20 : 00 - 21 : 00] - 坐地铁到机场
  • [21 : 00 - 23 : 00] - 办登机，等飞机
  • [23 : 00 - 4 : 00] - 新加坡樟宜 -> 上海浦东 (航班号HO1606)
  • [4 : 00 - 5 : 00] - 肥家继续睡觉
